# Руководство по настройке интеграции с календарём

Это руководство поможет вам настроить генерацию файлов календаря (.ics), публикацию через Discord-вебхук и запланированные мероприятия Discord в VRC Event Creator. Эти три функции полностью независимы — включайте любую комбинацию, подходящую вашему рабочему процессу.

---

## Обзор

VRC Event Creator предлагает три действия после создания мероприятия VRChat (вручную или автоматически). Каждое из них независимо включается для каждого шаблона и мероприятия:

- **«Создать приглашение календаря .ics»** — Создаёт стандартный файл календаря `.ics` с необязательными напоминаниями, автоматически сохраняемый в локальную директорию
- **«Отправить Discord Webhook»** — Публикует объявление в канал Discord через вебхук (с необязательным прикреплением файла `.ics`, если календарь тоже включён)
- **«Создать событие Discord»** — Создаёт запланированное мероприятие на вашем сервере Discord через бота

Когда несколько функций включены, они естественно дополняют друг друга:

| Событие Discord | Вебхук | Календарь (.ics) | Что происходит |
|---|---|---|---|
| ВКЛ | ВЫКЛ | ВЫКЛ | Создаётся только запланированное событие Discord |
| ВЫКЛ | ВКЛ | ВЫКЛ | Вебхук публикует карточку с деталями мероприятия |
| ВЫКЛ | ВЫКЛ | ВКЛ | Файл `.ics` сохраняется в локальную директорию |
| ВКЛ | ВКЛ | ВЫКЛ | Событие Discord создаётся + вебхук публикует ссылку на событие |
| ВКЛ | ВЫКЛ | ВКЛ | Событие Discord создаётся + `.ics` сохраняется |
| ВЫКЛ | ВКЛ | ВКЛ | Вебхук публикует карточку + `.ics` прикреплён, также сохранён |
| ВКЛ | ВКЛ | ВКЛ | Событие Discord + вебхук со ссылкой на событие + `.ics` прикреплён + сохранён |

---

## Шаг 1: Включить создание файлов календаря

1. Откройте **Настройки** > **Расширенные настройки**
2. Отметьте **«Включить создание файлов календаря»**

После этого переключатель **«Создать приглашение календаря .ics»** станет доступен в шаблонах и при создании мероприятий.

### Директория сохранения

Когда создание файлов календаря включено, файлы `.ics` всегда сохраняются в локальную директорию. Расположение по умолчанию — `Documents/VRC Event Creator .ics/`, создаётся при первом сохранении.

Файлы сохраняются как `{директория}/{Название группы}/{Название мероприятия - Дата}.ics`. Чтобы изменить расположение, используйте кнопку **Изменить** рядом с **Директория сохранения календаря** в **Настройки** > **Информация о приложении**.

---

## Шаг 2: Настроить Discord-вебхук (необязательно)

Вебхук публикует объявления в определённый канал Discord. Он не зависит от файлов календаря и мероприятий Discord — его можно использовать как вместе, так и без них.

1. В Discord нажмите правой кнопкой мыши на канал, куда хотите отправлять объявления
2. Нажмите **Редактировать канал** > **Интеграции** > **Вебхуки** > **Новый вебхук**
3. Скопируйте URL вебхука
4. В VRC Event Creator перейдите в **Настройки** > **Интеграция с Discord** > выберите свою группу
5. Отметьте **«Включить Webhook»** и вставьте URL вебхука
6. Нажмите **Проверить вебхук**, чтобы убедиться в работоспособности, затем **Сохранить**

Когда и вебхук, и календарь включены для мероприятия, файл `.ics` прикрепляется к публикации вебхука. Когда включён только вебхук (без календаря), вебхук публикует карточку с деталями мероприятия без прикрепления `.ics`.

Если также было создано запланированное событие Discord, сообщение вебхука содержит ссылку на событие Discord вместо карточки.

## Шаг 3: Настроить шаблоны

1. Перейдите в **Управление шаблонами** и отредактируйте (или создайте) шаблон
2. Во вкладке **Основы** вы увидите до трёх переключателей публикации (в зависимости от настроек):
   - **«Создать приглашение календаря .ics»** — отображается, когда создание файлов календаря включено
   - **«Создать событие Discord»** — отображается, когда бот Discord настроен для группы
   - **«Отправить Discord Webhook»** — отображается, когда URL вебхука настроен для группы
3. Включите нужные параметры для этого шаблона
4. Если календарь включён, вкладка **Расписание** показывает карточку **«Напоминания календаря .ics»**
5. Отметьте **«Включить напоминания календаря .ics»** и добавьте нужные интервалы напоминаний
6. Сохраните шаблон

Напоминания используют предустановленные интервалы, совместимые со всеми основными календарными приложениями: 5 мин, 10 мин, 15 мин, 30 мин, 1 час, 2 часа, 4 часа, 8 часов, 12 часов, 1 день, 2 дня, 1 неделя.

> **Примечание:** Некоторые календарные приложения (например, Outlook) используют только первое напоминание. Для лучшей совместимости самое длительное напоминание размещается первым. Google Calendar игнорирует пользовательские напоминания при импорте и использует ваши настройки уведомлений по умолчанию.

---

## Шаг 4: Создание мероприятий

При создании мероприятия (вручную или через автоматизацию):

- На шаге **Дата** отображается **«Создать приглашение календаря .ics»** (наследуется из шаблона, можно изменить)
- Ниже **«Включить напоминания календаря .ics»** позволяет настроить напоминания для каждого мероприятия
- На шаге **Детали** отображаются **«Создать событие Discord»** и **«Отправить Discord Webhook»** как отдельные переключатели
- Все настройки шаблона можно изменить для каждого мероприятия

## Часто задаваемые вопросы

### Какие календарные приложения поддерживают файлы .ics?

Все основные: Outlook, Apple Calendar, Google Calendar, Thunderbird и любое приложение, поддерживающее стандарт iCalendar.

### Работают ли напоминания во всех календарных приложениях?

Несколько напоминаний работают в Apple Calendar и Thunderbird. Outlook использует только первое напоминание. Google Calendar полностью игнорирует напоминания при импорте.

### Можно ли использовать вебхуки без файлов календаря?

Да. Вебхук публикует карточку с деталями мероприятия, даже когда создание файлов календаря отключено. Включите «Отправить Discord Webhook» в вашем шаблоне без включения «Создать приглашение календаря .ics».

### Можно ли использовать вебхуки без создания мероприятий Discord?

Да. Вебхук, мероприятия Discord и файлы календаря полностью независимы. Любая комбинация работает.

### Является ли URL вебхука конфиденциальным?

Да — любой, у кого есть URL вебхука, может отправлять сообщения в этот канал. Обращайтесь с ним как с паролем. Он шифруется и хранится локально с помощью защищённого хранилища вашей операционной системы.

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Файл .ics не генерируется | Убедитесь, что «Включить создание файлов календаря» включено в Расширенных настройках, и «Создать приглашение календаря .ics» отмечено в шаблоне или мероприятии |
| Вебхук не публикует | Проверьте URL вебхука через «Проверить вебхук» в настройках Discord. Убедитесь, что «Включить Webhook» включено для группы и «Отправить Discord Webhook» отмечено в шаблоне |
| Вебхук публикует, но без .ics | «Создать приглашение календаря .ics» также должно быть включено для мероприятия. Без этого вебхук публикует только карточку или ссылку на событие |
| Напоминания не работают в Outlook | Outlook поддерживает только первое напоминание. Приложение располагает самое длительное первым для совместимости |
| Напоминания не работают в Google Calendar | Google Calendar игнорирует пользовательские напоминания при импорте .ics. Настройте напоминания вручную после импорта |
| Файлы сохраняются в неправильном месте | Файлы сохраняются в `{директория}/{Название группы}/`. По умолчанию — `Documents/VRC Event Creator .ics/`. Можно изменить через Настройки > Информация о приложении |
