# Discord 연동 설정 가이드

이 가이드는 VRC Event Creator의 Discord 연동을 설정하는 과정을 안내합니다. 설정이 완료되면, VRChat 이벤트를 생성할 때 서버에 해당하는 **Discord 이벤트**가 자동으로 함께 생성됩니다.

---

## 개요

이 연동은 사용자가 직접 생성하고 관리하는 **Discord 봇**을 사용합니다. 필요한 권한은 **이벤트 만들기** 하나뿐입니다. 메시지를 읽거나 음성 채널에 참여하는 등의 다른 동작은 일절 하지 않습니다. 봇 토큰은 암호화되어 로컬에 저장되며 — 이벤트 생성 시 Discord API로 전송되는 것 외에는 어디에도 보내지지 않습니다.

각 VRChat 그룹은 하나의 Discord 서버에 연결할 수 있습니다. 같은 봇을 여러 그룹/서버에서 재사용하거나, 별도의 봇을 사용할 수도 있습니다.

---

## 1단계: Discord 애플리케이션 생성

1. [Discord Developer Portal](https://discord.com/developers/applications)로 이동합니다
2. 오른쪽 상단의 **"New Application"**을 클릭합니다
3. 이름을 입력하고 **Create**를 클릭합니다

## 2단계: 봇 생성

1. 왼쪽 사이드바에서 **"Bot"**을 클릭합니다
2. **"Reset Token"**을 클릭합니다 (토큰이 아직 보이면 **"Copy"** 클릭)
3. **토큰을 즉시 복사하세요** — 이후에는 다시 확인할 수 없습니다
4. Privileged Gateway Intents는 꺼둔 상태로 두세요 — 봇에 필요하지 않습니다

> **봇 토큰을 비공개로 유지하세요.** 토큰을 가진 사람은 누구나 봇으로 행동할 수 있습니다. 실수로 공유한 경우, Developer Portal에서 즉시 재설정하세요.

## 3단계: 서버에 봇 초대

1. 왼쪽 사이드바에서 **"OAuth2"**를 클릭합니다
2. **"OAuth2 URL Generator"**까지 스크롤합니다
3. **Scopes**에서 **`bot`**을 체크합니다
4. **Bot Permissions**에서 **`Create Events`**를 체크합니다
5. 하단에 생성된 URL을 복사하여 브라우저에서 열고, 서버를 선택한 뒤 승인합니다

봇이 멤버 목록에 나타나지만 오프라인 상태로 유지됩니다 — "실행" 중일 필요가 없습니다. 앱이 토큰을 사용하여 Discord API에 직접 요청합니다.

## 4단계: 서버 ID 확인

1. Discord에서 **사용자 설정** > **고급**으로 이동하여 **개발자 모드**를 활성화합니다
2. 서버 이름을 마우스 오른쪽 버튼으로 클릭하고 **"서버 ID 복사"**를 선택합니다

## 5단계: VRC Event Creator에서 설정

1. **설정** > **고급 옵션**을 열고 **"Discord 연동 활성화"**를 체크합니다
2. 연결할 VRChat 그룹을 선택하고, 봇 토큰과 서버 ID를 입력한 뒤 저장합니다
3. **"봇 토큰 확인"**을 사용하여 토큰이 정상 작동하는지 확인합니다

각 이벤트 템플릿의 Basics 탭에 **"Discord에 동기화"** 토글이 있습니다. Discord 연동이 활성화되어 있으면 기본적으로 켜져 있습니다. Discord에 게시하지 않을 이벤트는 이 토글을 꺼서 비활성화할 수 있습니다.

**Discord 동기화는 VRChat 이벤트 생성을 절대 차단하지 않습니다.** Discord 쪽에서 문제가 발생하더라도, VRChat 이벤트는 정상적으로 생성됩니다.

## 자주 묻는 질문

### 이미 가지고 있는 봇을 사용할 수 있나요?

네, 대상 서버에서 **이벤트 만들기** 권한이 있으면 사용할 수 있습니다.

### 여러 스태프가 이벤트를 생성하는 경우에는 어떻게 하나요?

이벤트를 생성하는 각 사용자의 기기에 봇 토큰이 필요합니다. 방법:
- **신뢰할 수 있는 스태프와 토큰 공유**
- **한 사람이 Discord 동기화를 담당**하고 나머지는 "Discord에 동기화"를 비활성화
- **스태프별로 별도의 봇 생성**

### 봇 토큰은 안전한가요?

봇 토큰은 운영 체제의 보안 저장소(Windows DPAPI / macOS Keychain / Linux Secret Service)를 사용하여 암호화되며 로컬에 저장됩니다. Discord API 외에는 어디로도 전송되지 않습니다.

### 앱에서 Discord 이벤트를 삭제할 수 있나요?

아니요, 앱은 이벤트를 생성하기만 합니다. Discord 이벤트는 Discord에서 직접 관리하세요.

## 문제 해결

| 문제 | 해결 방법 |
|---|---|
| "유효하지 않은 봇 토큰" | Developer Portal에서 토큰을 재설정하고 새 토큰을 붙여넣으세요 |
| "봇에 이벤트 생성 권한이 없습니다" | 이벤트 만들기 권한으로 봇을 다시 초대하거나, 서버 설정 > 역할에서 추가하세요 |
| "Discord 서버를 찾을 수 없습니다" | 서버 ID를 다시 확인하세요 (서버 우클릭 > 서버 ID 복사) |
| "Discord 요청 제한에 도달했습니다" | 1분 정도 기다린 후 다시 시도하세요 |
| VRChat에서는 이벤트가 생성되었지만 Discord에는 생성되지 않음 | "Discord에 동기화"가 활성화되어 있고 해당 그룹에 유효한 봇 토큰 + 서버 ID가 설정되어 있는지 확인하세요 |
