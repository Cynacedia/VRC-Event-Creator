# Discord 集成设置指南

本指南将引导你完成 VRC Event Creator 的 Discord 集成设置。配置完成后，创建 VRChat 活动时会自动在你的服务器中创建对应的 **Discord 活动**。

---

## 概述

该集成使用一个由你自行创建和管理的 **Discord 机器人**。它只需要一个权限：**创建活动**。它不会读取消息、加入语音频道，也不会执行任何其他操作。你的机器人令牌会被加密并存储在本地 — 仅在创建活动时发送至 Discord API。

每个 VRChat 群组可以关联一个 Discord 服务器。你可以在多个群组/服务器中复用同一个机器人，也可以使用不同的机器人。

---

## 第 1 步：创建 Discord 应用

1. 前往 [Discord Developer Portal](https://discord.com/developers/applications)
2. 点击右上角的 **"New Application"**
3. 输入名称，然后点击 **Create**

## 第 2 步：创建机器人

1. 点击左侧边栏中的 **"Bot"**
2. 点击 **"Reset Token"**（如果令牌仍可见，则点击 **"Copy"**）
3. **立即复制令牌** — 之后将无法再次查看
4. 保持 Privileged Gateway Intents 关闭 — 机器人不需要这些权限

> **请妥善保管你的机器人令牌。** 任何持有令牌的人都可以以你的机器人身份进行操作。如果不慎泄露，请立即在 Developer Portal 中重置。

## 第 3 步：将机器人邀请到你的服务器

1. 点击左侧边栏中的 **"OAuth2"**
2. 向下滚动到 **"OAuth2 URL Generator"**
3. 在 **Scopes** 下勾选 **`bot`**
4. 在 **Bot Permissions** 下勾选 **`Create Events`**
5. 复制底部生成的 URL，在浏览器中打开，选择你的服务器并授权

机器人会出现在成员列表中，但显示为离线 — 它不需要"运行"。应用程序通过令牌直接调用 Discord API。

## 第 4 步：获取服务器 ID

1. 在 Discord 中，前往 **用户设置** > **高级** 并启用 **开发者模式**
2. 右键点击服务器名称，选择 **"复制服务器 ID"**

## 第 5 步：在 VRC Event Creator 中配置

1. 打开 **设置** > **高级选项** > 勾选 **"启用 Discord 集成"**
2. 选择要关联的 VRChat 群组，输入机器人令牌和服务器 ID，然后保存
3. 使用 **"验证机器人令牌"** 确认令牌有效

每个活动模板的 Basics 标签页中都有一个 **"同步到 Discord"** 开关。当 Discord 集成处于启用状态时，此开关默认开启。你可以为不想发布到 Discord 的活动关闭此开关。

**Discord 同步绝不会阻碍 VRChat 活动的创建。** 即使 Discord 端出现问题，你的 VRChat 活动仍会正常创建。

## 常见问题

### 我可以使用已有的机器人吗？

可以，只要该机器人在目标服务器中拥有 **创建活动** 权限即可。

### 如果多名工作人员共同创建活动怎么办？

每位创建活动的成员都需要在自己的设备上拥有机器人令牌。解决方案：
- **与可信的工作人员共享令牌**
- **由一人负责 Discord 同步**，其他人关闭"同步到 Discord"
- **为每位工作人员创建单独的机器人**

### 我的机器人令牌安全吗？

你的机器人令牌通过操作系统的安全存储（Windows DPAPI / macOS 钥匙串 / Linux Secret Service）进行加密，并存储在本地。它仅会被发送至 Discord API。

### 我可以从应用中删除 Discord 活动吗？

不可以，应用只负责创建活动。请直接在 Discord 中管理 Discord 活动。

---

## 故障排除

| 问题 | 解决方案 |
|---|---|
| "无效的机器人令牌" | 在 Developer Portal 中重置令牌并粘贴新令牌 |
| "机器人没有创建活动的权限" | 使用"创建活动"权限重新邀请机器人，或通过服务器设置 > 身份组进行添加 |
| "未找到 Discord 服务器" | 请仔细检查服务器 ID（右键点击服务器 > 复制服务器 ID） |
| "已达到 Discord 速率限制" | 请等待一分钟后重试 |
| 活动已在 VRChat 中创建，但未在 Discord 中创建 | 请检查"同步到 Discord"是否已启用，以及该群组是否已配置有效的机器人令牌和服务器 ID |
