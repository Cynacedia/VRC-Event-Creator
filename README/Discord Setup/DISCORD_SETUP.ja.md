# Discord連携セットアップガイド

VRC Event CreatorのDiscord連携を設定する手順です。設定が完了すると、VRChatイベントを作成するたびにDiscordサーバーに**Discordイベント**が自動的に作成されます。

---

## 概要

あなたが作成・管理する**Discord Bot**を使用します。必要な権限は**イベントを作成**のみです。メッセージの読み取りやボイスチャンネルへの参加は行いません。BotトークンはOS標準の暗号化でローカルに保存され、Discordイベント作成時にDiscord APIへ送信される以外、どこにも送信されません。

VRChatグループごとに1つのDiscordサーバーをリンクできます。同じBotを複数のグループやサーバーで使い回すことも、別々のBotを使うことも可能です。

---

## ステップ1：Discordアプリケーションを作成する

1. [Discord Developer Portal](https://discord.com/developers/applications)を開く
2. 右上の「**New Application**」をクリック
3. 名前を入力し「**Create**」をクリック

## ステップ2：Botを作成する

1. 左メニューの「**Bot**」をクリック
2. 「**Reset Token**」をクリック（初回は「**Copy**」で直接コピー可能）
3. **トークンをすぐにコピー** — 一度画面を離れると再表示できません
4. Privileged Gateway Intentsはオフのままでよい — Botには不要

> **Botトークンは秘密にしてください。**トークンを知っている人は誰でもBotとして操作できます。万が一共有してしまった場合は、開発者ポータルで直ちにリセットしてください。

## ステップ3：Botをサーバーに招待する

1. 左メニューの「**OAuth2**」をクリック
2. 「**OAuth2 URL Generator**」まで下にスクロール
3. 「**Scopes**」で「**bot**」にチェック
4. 「**Bot Permissions**」で「**Create Events**」（イベントを作成）にチェック
5. 生成されたURLをコピーし、ブラウザで開いてサーバーを選択して認証

Botはメンバーリストに表示されますがオフラインのままです — 常時稼働は不要です。アプリがトークンを使ってDiscord APIを直接呼び出します。

## ステップ4：サーバーIDを取得する

1. Discordの**ユーザー設定** > **詳細設定**で**開発者モード**を有効にする
2. サーバー名を右クリックし「**サーバーIDをコピー**」をクリック

## ステップ5：VRC Event Creatorで設定する

1. **設定** > **詳細設定** > 「**Discord連携を有効にする**」にチェック
2. リンクしたいVRChatグループを選択し、BotトークンとサーバーIDを入力して保存
3. 「**Botトークンを検証**」でトークンの動作を確認

各テンプレートの基本設定タブに「**Discordに同期**」トグルがあります。Discord連携が有効な場合、デフォルトでオンになります。Discordに投稿したくないイベントはオフに切り替えてください。

**Discord同期がVRChatイベントの作成を妨げることはありません。**Discord側で何か問題が起きても、VRChatイベントは通常どおり作成されます。

## よくある質問

### 既存のBotを使えますか？

対象サーバーで「**Create Events**」（イベントを作成）権限があれば使えます。

### イベント作成を複数のスタッフで分担する場合は？

各スタッフが自分のPCにBotトークンを設定する必要があります：
- **信頼できるスタッフとトークンを共有する**
- **1人がDiscord同期を担当し**、他のスタッフは「Discordに同期」をオフにする
- スタッフごとに**別のBotを作成する**

### Botトークンは安全ですか？

OSの安全なストレージ（Windows DPAPI / macOSキーチェーン / Linux Secret Service）で暗号化されローカルに保存されます。Discord API以外にどこにも送信されません。

### アプリからDiscordイベントを削除できますか？

できません。アプリはイベントの作成のみ行います。DiscordイベントはDiscord上で直接管理してください。

## トラブルシューティング

| 問題 | 解決方法 |
|---|---|
| 「無効なBotトークン」 | 開発者ポータルでトークンをリセットし新しいトークンを貼り付け |
| 「Botにイベント作成権限がありません」 | Create Events権限を付けてBotを再招待、またはサーバー設定 > ロールで権限を追加 |
| 「Discordサーバーが見つかりません」 | サーバーIDを確認（サーバー名を右クリック > サーバーIDをコピー） |
| 「Discordのレート制限に達しました」 | 1分ほど待ってから再試行 |
| VRChatイベントは作成されるがDiscordに反映されない | 「Discordに同期」が有効であること、グループにBotトークンとサーバーIDが保存されていることを確認 |
