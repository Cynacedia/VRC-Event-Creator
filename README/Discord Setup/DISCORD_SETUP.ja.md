# Discord連携セットアップガイド

このガイドでは、VRC Event CreatorのDiscord連携を設定する手順を説明します。設定が完了すると、VRChatイベントを作成するたびに、Discordサーバーの**イベント**が自動的に作成されます。サイドバーに表示され、通知が届き、メンバーが「興味あり」を押せるイベントです。

---

## 概要

この連携では、あなたが作成・管理する**Discord Bot**を使用します。Botに必要な権限は**イベントを作成**のみです。メッセージの読み取りやボイスチャンネルへの参加などは一切行いません。Botトークンはお使いのPCにOS標準の暗号化で保存され、Discordへのイベント作成時以外にどこにも送信されません。

VRChatグループごとに1つのDiscordサーバーをリンクできます。同じBotを複数のグループやサーバーで使い回すことも、グループごとに別のBotを使うことも可能です。

---

## ステップ1：Discordアプリケーションを作成する

1. [Discord Developer Portal](https://discord.com/developers/applications)（Discord開発者ポータル）を開きます
2. 右上の「**New Application**」（新しいアプリケーション）をクリックします
3. 名前を入力し（例：「VRCイベントBot」）、「**Create**」（作成）をクリックします

## ステップ2：Botを作成する

1. 作成したアプリケーションの画面で、左メニューの「**Bot**」をクリックします
2. 「**Token**」セクションにある「**Reset Token**」（トークンをリセット）をクリックします（初回の場合は「**Copy**」で直接コピーできます）
3. **トークンをすぐにコピーしてください** — 一度画面を離れると再表示できません。紛失した場合は、開発者ポータルでリセットする必要があります
4. 「Privileged Gateway Intents」（特権インテント）はすべてオフのままで構いません — このBotにはインテントは不要です

> **Botトークンは秘密にしてください。** トークンを知っている人は誰でもそのBotとして操作できます。万が一共有してしまった場合は、開発者ポータルで直ちにリセットしてください。

## ステップ3：Botをサーバーに招待する

1. 開発者ポータルで、左メニューの「**OAuth2**」をクリックします
2. 「**OAuth2 URL Generator**」まで下にスクロールします
3. 「**Scopes**」（スコープ）で「**bot**」にチェックを入れます
4. 「**Bot Permissions**」（Botの権限）で「**イベントを作成**」（Create Events）にチェックを入れます — これが唯一必要な権限です
5. ページ下部に生成されたURLをコピーします
6. そのURLをブラウザで開き、Botを追加したいDiscordサーバーを選択します
7. 「**認証**」をクリックします

Botはサーバーのメンバーリストに表示されますが、常にオフラインのままです（Botを常時稼働させる必要はありません — VRC Event CreatorがBotトークンを使って直接Discord APIを呼び出します）。

## ステップ4：サーバーIDを取得する

1. Discordで**ユーザー設定**（ユーザー名の横にある歯車アイコン）を開きます
2. 「**詳細設定**」を開き、「**開発者モード**」を有効にします
3. 設定を閉じ、サーバーリストで**サーバー名を右クリック**します
4. 「**サーバーIDをコピー**」をクリックします

サーバーIDは `123456789012345678` のような長い数字です。

## ステップ5：VRC Event Creatorで設定する

1. VRC Event Creatorを開き、**設定**画面に移動します
2. **詳細設定**まで下にスクロールします
3. 「**Discord連携を有効にする**」にチェックを入れます
4. リンクしたい**VRChatグループ**を選択します
5. 「Botトークン」欄に**Botトークン**を貼り付けます
6. 「サーバーID」欄に**サーバーID**を貼り付けます
7. 「**Botトークンを検証**」をクリックして、トークンが正しいか確認します — DiscordのAPIに接続してBotの存在を検証します
8. 「**保存**」をクリックします

## ステップ6：テンプレートごとの同期設定

各イベントテンプレートには、テンプレート編集画面（基本設定タブ）に「**Discordに同期**」トグルがあります。Discord連携が有効な場合に表示されます。Discordに投稿したくないテンプレートではこれをオフにしてください。

---

## 仕組み

VRChatイベントを作成すると（手動でも自動でも）、アプリは以下の処理を行います：

1. 通常通りVRChatイベントを作成します
2. そのグループでDiscord連携が有効かどうか確認します
3. テンプレートの「Discordに同期」が有効かどうか確認します
4. 以下の内容でDiscordイベントを作成します：
   - **タイトル** — VRChatイベントの名前
   - **説明** — VRChatイベントの説明
   - **開始・終了時刻** — VRChatイベントと同じ
   - **場所** — 「VRChat」
   - **カバー画像** — VRChatイベントの画像（設定されている場合）
5. Discordイベントが作成されたことを通知します

**Discord同期がVRChatイベントの作成を妨げることはありません。** Discord APIが利用できない場合やBotトークンが無効な場合でも、VRChatイベントは正常に作成されます — Discord側のエラー通知が表示されるだけです。

## よくある質問

### 既存のBotを使えますか？

はい、対象サーバーで「**イベントを作成**」権限を持っていれば使えます。同じBotトークンを複数のグループやサーバーで使用できます。

### 同じBotを複数のグループで使えますか？

はい。各グループに同じBotトークンを貼り付け、それぞれのサーバーIDを設定してください。サーバーIDがイベントの投稿先を決定します。

### イベント作成を複数のスタッフで分担している場合は？

イベントを作成する各スタッフが、自分のPCにBotトークンを設定する必要があります。方法はいくつかあります：
- **信頼できるスタッフとトークンを共有する**（同じBot権限を持つことになります）
- **1人がDiscord同期を担当し**、他のスタッフは「Discordに同期」をオフにしてイベントを作成する
- スタッフごとに**別のBotを作成する**（誰がイベントを作成したか区別したい場合）

### Botにはどんな権限が必要ですか？

「**イベントを作成**」のみです。メッセージの読み取りやサーバーの管理など、他の権限は一切不要です。

### Botトークンは安全ですか？

BotトークンはOSの安全なストレージ（WindowsのDPAPI / macOSのキーチェーン / LinuxのSecret Service）を使って暗号化され、プロファイルデータにローカル保存されます。DiscordのAPIへのイベント作成時以外、どこにも送信されません。

### Botがオフラインと表示されますが、正常ですか？

はい。このBotはDiscordへの常時接続を維持しません。VRC Event Creatorが必要な時にBotトークンを使ってAPIを直接呼び出す仕組みです。Botは常にオフライン表示になりますが、問題ありません。

### Discord同期が失敗した場合は？

VRChatイベントは正常に作成されます。何が問題だったか（無効なトークン、権限不足、レート制限など）を説明するエラー通知が表示されます。問題を修正すれば、次回のイベントから正常に同期されます。

### アプリからDiscordイベントを削除できますか？

現時点ではできません。Discordイベントは、Discord上で直接管理してください。このアプリはイベントの作成のみを行います。

## トラブルシューティング

| 問題 | 解決方法 |
|---|---|
| 「無効なBotトークン」と表示される | 開発者ポータルでトークンをリセットし、新しいトークンを貼り付けてください |
| 「Botにイベント作成権限がありません」と表示される | 「イベントを作成」権限を付けてBotを再招待するか、サーバー設定 > ロールから権限を追加してください |
| 「Discordサーバーが見つかりません」と表示される | サーバーIDを再確認してください（サーバー名を右クリック > サーバーIDをコピー） |
| 「Discordのレート制限に達しました」と表示される | 1分ほど待ってから再度お試しください。短時間に大量のイベントを作成すると発生します |
| Botトークンの検証が失敗する | トークンを余分なスペースなく完全にコピーしたか確認してください |
| VRChatイベントは作成されるがDiscordに反映されない | テンプレートの「Discordに同期」が有効であること、グループに有効なBotトークンとサーバーIDが保存されていることを確認してください |
