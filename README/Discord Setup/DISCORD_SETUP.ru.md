# Руководство по настройке интеграции с Discord

Это руководство поможет вам настроить интеграцию с Discord для VRC Event Creator. После настройки при создании мероприятия в VRChat автоматически будет создано соответствующее **мероприятие в Discord** на вашем сервере.

---

## Обзор

Интеграция использует **Discord-бота**, которого вы сами создаёте и контролируете. Ему нужно только одно разрешение: **Создавать мероприятия**. Бот не читает сообщения, не подключается к голосовым каналам и не выполняет никаких других действий. Токен бота шифруется и хранится локально — он отправляется только в API Discord при создании мероприятий.

Каждую группу VRChat можно привязать к одному серверу Discord. Вы можете использовать одного и того же бота для нескольких групп/серверов или создать отдельных ботов.

---

## Шаг 1: Создайте приложение Discord

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Нажмите **«New Application»** в правом верхнем углу
3. Введите название и нажмите **Create**

## Шаг 2: Создайте бота

1. Нажмите **«Bot»** в левой боковой панели
2. Нажмите **«Reset Token»** (или **«Copy»**, если токен ещё отображается)
3. **Скопируйте токен немедленно** — повторно просмотреть его не получится
4. Оставьте Privileged Gateway Intents отключёнными — боту они не нужны

> **Храните токен бота в тайне.** Любой, кто получит токен, сможет действовать от имени вашего бота. Если вы случайно его раскрыли, немедленно сбросьте токен в Developer Portal.

## Шаг 3: Пригласите бота на ваш сервер

1. Нажмите **«OAuth2»** в левой боковой панели
2. Прокрутите до раздела **«OAuth2 URL Generator»**
3. В разделе **Scopes** отметьте **`bot`**
4. В разделе **Bot Permissions** отметьте **`Create Events`**
5. Скопируйте сгенерированную ссылку внизу, откройте её в браузере, выберите свой сервер и подтвердите авторизацию

Бот появится в списке участников, но останется оффлайн — ему не нужно быть «запущенным». Приложение напрямую обращается к API Discord, используя токен.

## Шаг 4: Получите ID вашего сервера

1. В Discord откройте **Настройки пользователя** > **Расширенные** и включите **Режим разработчика**
2. Нажмите правой кнопкой мыши на название сервера и выберите **«Копировать ID сервера»**

## Шаг 5: Настройте в VRC Event Creator

1. Откройте **Настройки** > **Расширенные параметры** > отметьте **«Включить интеграцию с Discord»**
2. Выберите группу VRChat, которую хотите привязать, введите токен бота и ID сервера, затем сохраните
3. Используйте **«Проверить токен бота»**, чтобы убедиться, что токен работает

В каждом шаблоне мероприятия на вкладке Basics есть переключатель **«Синхронизировать с Discord»**. По умолчанию он включён, когда интеграция с Discord активна. Вы можете отключить его для мероприятий, которые не нужно публиковать в Discord.

**Синхронизация с Discord никогда не блокирует создание мероприятий в VRChat.** Если на стороне Discord что-то пойдёт не так, ваше мероприятие в VRChat всё равно будет создано в обычном режиме.

## Часто задаваемые вопросы

### Можно ли использовать уже существующего бота?

Да, если у него есть разрешение **Создавать мероприятия** на целевом сервере.

### Что делать, если несколько сотрудников создают мероприятия?

Каждому, кто создаёт мероприятия, нужен токен бота на своём компьютере. Варианты:
- **Поделитесь токеном** с доверенными сотрудниками
- **Назначьте одного ответственного за синхронизацию с Discord**, а остальные отключат «Синхронизировать с Discord»
- **Создайте отдельных ботов** для каждого сотрудника

### Безопасен ли мой токен бота?

Ваш токен бота шифруется с помощью защищённого хранилища операционной системы (Windows DPAPI / macOS Keychain / Linux Secret Service) и хранится локально. Он отправляется только в API Discord.

### Можно ли удалять мероприятия Discord из приложения?

Нет, приложение только создаёт их. Управляйте мероприятиями Discord непосредственно в Discord.

## Устранение неполадок

| Проблема | Решение |
|---|---|
| «Недействительный токен бота» | Сбросьте токен в Developer Portal и вставьте новый |
| «У бота нет разрешения на создание мероприятий» | Повторно пригласите бота с разрешением «Создавать мероприятия» или добавьте его через Настройки сервера > Роли |
| «Сервер Discord не найден» | Проверьте ID сервера (правый клик по серверу > Копировать ID сервера) |
| «Превышен лимит запросов Discord» | Подождите минуту и попробуйте снова |
| Мероприятия созданы в VRChat, но не в Discord | Убедитесь, что «Синхронизировать с Discord» включён, а у группы указаны действительный токен бота и ID сервера |
